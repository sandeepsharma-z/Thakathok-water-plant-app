# Thaka Thok Water App - Project Scope

## Project Overview

Thaka Thok Water App is being developed to help the water plant business manage daily operations digitally.

The main purpose of this system is to manage:

- Customer orders
- Pending payments
- Empty jar/can returns
- Delivery staff updates
- Admin approval
- Monthly dues/reporting
- Customer order eligibility
- Delivery route/order visibility

## Included Modules

### 1. Customer App

The customer app will allow customers to:

- Login using mobile number OTP
- Manage basic profile details
- Place water jar/can orders
- Track order status
- View pending payment
- View pending empty jar/can count
- View account/order eligibility status
- Place next order only when previous dues/cans are cleared

### 2. Admin Dashboard

The admin dashboard will allow admin/owner to:

- Manage customers
- Manage delivery areas
- Manage products and jar/can rates
- Manage orders
- Assign orders to delivery staff
- Track pending payments
- Track empty jar/can returns
- View customer-wise order/payment history
- Download monthly dues report
- Mark order as "All Done"
- Control customer eligibility for next order

### 3. Delivery Staff Panel

The delivery staff panel will allow staff to:

- Login securely
- View assigned orders
- View delivery route/order details
- Update delivery status
- Record collected payment
- Record empty jar/can returns
- Add delivery remarks if required
- Sync updates with admin dashboard

## Core Business Rule

If a customer has pending payment or pending empty jar/can return from a previous order, the customer will not be allowed to place another order until the dues are cleared and admin verifies the order.

The order will be completed only after admin marks it as "All Done".

## Verification Flow
