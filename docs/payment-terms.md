# Payment Terms

## Approved Project Cost

Total Project Cost: ₹75,000 one-time.

## Approved Payment Structure

1. 30% Advance: ₹22,500
2. 40% After Working Demo: ₹30,000
3. 30% Before Final Source Code Handover and AWS Deployment: ₹22,500

## Advance Payment

Advance amount of ₹22,500 will be released after the initial project setup commit is pushed to the client-owned GitHub repository as agreed.

Official project development will start only after advance payment confirmation/UTR receipt and required access/API details are shared by the client side.

Any timeline, milestone delivery period, weekly review cycle, and active development work will begin only after the advance payment is received and confirmed.

## Second Milestone

40% payment of ₹30,000 will be released within 2 days after live working demo of Admin Panel + Customer App + Delivery Panel on test server.

Demo will include:

- Customer order placement
- Admin order visibility
- Delivery boy route/order visibility

## Final Milestone

Final 30% payment of ₹22,500 will be released before final source code handover, complete access documentation, and AWS deployment completion.

## No Monthly Charges

There will be no monthly charges from Solvinex side.

## No Per-Transaction Charges

There will be no per-transaction charges from Solvinex side.

## Third-Party Charges

Any third-party charges like AWS hosting, SMS OTP, domain, Play Store, Google Maps, or payment gateway charges will be paid directly from the client-owned account only.
